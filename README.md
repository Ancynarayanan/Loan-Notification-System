# Loan Notification System

This is a Django-based web application for sending SMS and WhatsApp notifications for loan and document registration.

## Features
- Admin Login
- Team Login
- Send SMS notification
- Send WhatsApp notification
- Loan and document form submission
- Branch selection

## Technologies Used
- Python
- Django
- Twilio API
- HTML
- CSS
- JavaScript

## Project Structure
- twilioproject – Main Django project
- twilioapp – Application for SMS and messaging
- templates – HTML pages
- static – Images and static files
